COMMAND    			DESCRIPTION
_______________________________________________________________________________________
	     MOVEMENT
---------------------------------------------------------------------------------------    
	
cd 				change to my home directory
cd ..				move back
cd ""				change to "" directory	

_______________________________________________________________________________________
	     RETURNS
--------------------------------------------------------------------------------------- 

ls				list content
ls -la				list hidden files and content
tree 				interactive list
	
pwd				print working directory
which				path of a program 
xdg-open 			open a file with a default program 
	
head 				returns first lines of a file 
cat 				show file
tail 				returns last lines of a file

_______________________________________________________________________________________
	     CREATE
---------------------------------------------------------------------------------------
	
mkdir				creates directories 
touch 		  		create empty files
mv				rename and move files
cp 				copies files and directories (-r)
rm				remove files and directories (-r)
